Kindergarten Finder

×
  • July-August 2018
  • ~100 hours
  • Check result!
  • Source code
Technologies
  • React
  • Sass
  • Google Maps API
  • Facebook API

Details

Final project for FEND Degree aligned with the real needs, and hopefully the result can be useful for some people. ESPA is an European funding program which, among other, provides preschool sponsorship for parents with low income in Greece. Every year the catalogues of nurseries and kindergartens are published on the website as pdf, and as a parent you spend quite some time, googling every address in your area in order to find if it's located good for you. The backend script parses this PDF files and geocodes all the addresses, exporting a nice-formatted json with only useful information. This json is loaded on front end to Google Map, showing all the locations together with data on the ages of children, operation hours, etc. The ap also attempts to search for the page of kindergarten on Facebook, and if it exists, shows some useful information from it.

Merlin2Excel

×
  • March 2018
  • ~20 hours
  • Source code
Technologies
  • Node.js
  • Moment.js
  • xml2js
  • Excel4node

Details

A simple Node.js tool, created to avoid routine tasks at my office job. It's purpose is so specific, that it's probably useless for anyone else, although it can work with any type of data. The script parses xml file, exported from Merlin project, and creates a nice client-friendly Excel table. It also preserves the formatting (e.g. one task - one line), and applies colors to each type of activity, based on simple dictionary.
